# Email Metrics Analytics Dashboard

A comprehensive analytics dashboard for email marketing campaigns with AI-powered insights.

## Features

- 📊 **Interactive Analytics Dashboard** - Visualize email campaign performance
- 🤖 **AI-Powered Insights** - OpenAI GPT-4o integration for intelligent analysis
- 📈 **Performance Metrics** - Track open rates, click rates, conversion metrics
- 🔄 **Flow Analysis** - Analyze email sequences and automation performance
- 👥 **Audience Segmentation** - Understand subscriber behavior patterns
- ⏰ **Timing Optimization** - Best time and day analysis

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Set up your OpenAI API key:
```bash
cp .env.example .env
# Add your VITE_OPENAI_API_KEY to .env
```

3. Start development server:
```bash
npm run dev
```
